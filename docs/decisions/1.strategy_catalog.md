## 전략별 상세

### 1. F&G 역추세 (폐기)

**무엇인가?**
공포탐욕지수(Fear & Greed)가 25 이하(극공포)일 때 매수하고, 75 이상(극탐욕)일 때 매도하는 전략. "남들이 두려워할 때 사라"는 역발상 투자.

**어떻게 하는가?**
- 매수: F&G 지수 <= 25 → 다음날 시가에 BTC 매수
- 매도: F&G 지수 >= 75 → 다음날 시가에 매도
- 추가 시도: MA200 필터, 쿨다운(손절 후 120일 재진입 금지)

**결과**
- 인샘플(2023~2025): Sharpe 1.176, 승률 75% — 좋아 보였음
- **아웃오브샘플(2025~2026): Sharpe -1.269, MDD -43.4%, 승률 33%** — 완전 실패
- 원인: 인샘플 기간에 BTC 대형 상승장 포함 → 과적합. 실제로는 "극공포"가 더 큰 하락의 시작일 수 있음
- MA200 필터 추가 시 신호 대부분 소멸(구조적 충돌)

**현재 적용**: 폐기. 업비트 현물(롱만 가능)에서 역추세는 구조적 한계.

### 2. DC+ATR (코드 보존, 미사용)

**무엇인가?**
돈치안 채널(50일) 상단을 돌파하면 매수, ATR 기반 트레일링스탑으로 청산하는 클래식 추세추종.

**어떻게 하는가?**
- 매수: 오늘 종가 > 최근 50일 최고가 → "추세 시작"으로 보고 진입
- 매도: 보유 중 최고가에서 ATR(14) x 3.0만큼 떨어지면 → "추세 끝"으로 보고 청산
- ATR은 최근 14일간 가격 변동 평균. 변동성이 크면 스탑이 넓어지고, 작으면 좁아짐

**결과**
- OOS Sharpe 1.049, MDD -18.4%, 7거래, 승률 57.1%
- 안정적이지만 거래 빈도가 매우 낮음 (2년간 7번)
- Composite의 기반이 됨

**현재 적용**: Composite로 발전시켜서 대체됨. 코드는 `services/strategies/advanced.py`에 보존.

### 3. RSI+EMA (관찰용)

**무엇인가?**
RSI(상대강도지수)로 모멘텀을 확인하고, EMA(지수이동평균)로 장기 추세를 필터링하는 전략.

**어떻게 하는가?**
- 매수 조건: 종가 > EMA(150) **그리고** RSI(10) > 50
  - EMA(150) 위 = "장기적으로 상승추세"
  - RSI(10) > 50 = "단기 모멘텀도 상승 전환"
- 매도 조건: RSI(10) < 45 → 모멘텀 약화 시 청산

**결과**
- OOS Sharpe 1.040, MDD -14.9%, 15거래, 승률 46.7%
- DC+ATR보다 거래 빈도가 2배 높고 MDD가 작음
- 단점: 횡보장에서 잦은 진입/청산으로 소액 손실 누적

**현재 적용**: 관찰용으로 daily_check.py에서 신호만 모니터링. 실전 미투입.

### 4. 앙상블 (미사용)

**무엇인가?**
DC+ATR과 RSI+EMA 두 전략의 신호를 투표로 결합. 둘 중 하나라도 매수 신호를 내면 진입.

**어떻게 하는가?**
- DC 돌파 OR RSI 매수 → 진입
- 두 전략의 매도 신호 중 먼저 오는 것으로 청산

**결과**
- OOS Sharpe 0.985, MDD -23.9% — 엄격 기준(Sharpe 1.0+) 미달
- 거래 빈도는 늘었지만 MDD도 함께 증가

**현재 적용**: 미사용. Composite가 더 정교한 복합 방식으로 대체.

### 5. 변동성 레짐 (미사용)

**무엇인가?**
시장 변동성이 높을 때/낮을 때 스탑 거리를 자동으로 조절하는 전략.

**어떻게 하는가?**
- ATR/종가 비율로 변동성 레짐 판별 (60일 기준)
- 고변동 시: ATR 배수 4.0 (스탑 넓게 = 흔들림에 안 빠짐)
- 저변동 시: ATR 배수 2.0 (스탑 좁게 = 빠른 탈출)
- 진입은 DC(50) 돌파 동일

**결과**
- OOS Sharpe 0.943, MDD -21.5% — 기준 미달
- 아이디어는 좋으나 DC(50)과 큰 차이 없음

**현재 적용**: 미사용. 다만 Composite에 "적응형 스탑" 개념이 일부 반영됨.

### 6. MTF — 멀티타임프레임 (미사용)

**무엇인가?**
장기 추세(EMA 200일)와 단기 돌파(DC 30일)를 이중 확인하는 전략.

**어떻게 하는가?**
- 1차 필터: 종가 > EMA(200) → "큰 그림에서 상승추세인가?"
- 2차 진입: 종가 > DC(30) 상단 → "단기 돌파도 발생했는가?"
- 둘 다 만족해야 매수. 청산은 ATR(14) x 3.0 트레일링.

**결과**
- OOS Sharpe 0.767, MDD -19.7%, 6거래, 승률 50%
- 필터가 너무 보수적 → 진입 기회 극소

**현재 적용**: 미사용.

### 7. 거래량 확인 Volume Confirmation (미사용)

**무엇인가?**
DC 돌파 + 거래량 급증을 동시에 확인해야 진입하는 전략. "가격만 돌파한 건 가짜일 수 있다. 거래량도 터져야 진짜다."

**어떻게 하는가?**
- 매수: 종가 > DC(50) 상단 **그리고** 거래량 > 20일 평균 거래량 x 1.5
- 매도: ATR(14) x 3.0 트레일링스탑

**결과**
- OOS Sharpe 1.042, MDD -21.1%, 6거래, 승률 66.7%
- Sharpe은 통과했으나 MDD -21.1%이 기준(-20%) 근접 초과

**현재 적용**: 미사용. 거래량 필터 아이디어는 Composite에 흡수됨.

### 8. Composite (실전 메인 전략)

**무엇인가?**
DC+ATR, RSI, 거래량 확인, 적응형 스탑을 하나로 합친 복합 전략. 이 프로젝트의 **최종 진화형**.

**어떻게 하는가?**
- **진입 조건** (모두 충족):
  1. 종가 > DC(20) 상단 (20일 최고가 돌파) ← 필수
  2. RSI(10) > 50 **또는** 거래량 > 20일 평균 x 1.5 ← 둘 중 하나
- **청산**: 적응형 트레일링스탑
  - 변동성(ATR/종가)의 백분위 순위에 따라 배수를 2.0~4.0 사이에서 자동 결정
  - 변동성 높으면 → 스탑 넓게 (배수 4.0), 낮으면 → 스탑 좁게 (배수 2.0)
- **운용**: 최대 5종목 동시 보유, 종목당 가용자금의 19%(=95%/5)

**결과**
- OOS(2024~2026) Sharpe **1.280**, MDD -26.4%, 11거래, 수익률 +101%
- 하락장 실전(2026.03~04, 4거래): 승률 50%, 수익률 -2.8% vs Buy&Hold -32.1%
  - → Buy&Hold 대비 **+29.3%p 방어** — 하락장에서 자본 보전 능력 입증
- 현재 실전: 9거래, 1승 8패, 누적 -16.9% (극공포장 F&G 10 수준에서 운용 중)

**현재 적용**: **실전 운용 중**
- `config.py`: STRATEGY="composite", dc_period=20, ATR x3.0
- 실시간 웹소켓 모니터링, 최대 5포지션
- 현재 4종목 보유 (ONT +38.8%, RVN +1.0%, TRX -0.8%, CHZ -6.1%)

**DC 50 vs DC 20 차이**:
- DC(50): 50일 최고가 돌파 시 진입. 보수적. 연 3~4회 거래.
- DC(20): 20일 최고가 돌파 시 진입. 공격적. 연 5~6회 거래. Sharpe 더 높지만 MDD도 큼.
- 03-29에 DC50→DC20으로 전환 결정 (하락장에서 거래 빈도 확보 목적)

### 9. Daytrading (실패 중단)

**무엇인가?**
4시간봉 기반 단타. 짧은 타임프레임에서 빈번하게 매매하여 수익을 쌓는 전략.

**어떻게 하는가?**
- 매수: DC(15) 상단 돌파 + SMA(50) 위 + 거래량 2.5배 + 양봉 (4가지 동시 충족)
- 매도: +2% 트레일링 / -1.5% 손절 / 48시간 시간제한
- 백테스트: 2년간 726거래, 승률 38%, 건당 +0.91%

**결과**
- 백테스트는 괜찮아 보였으나 **실전 2일 만에 참패**
- 실전: 8건 중 1승 7패 (12.5%), 6연패, 평균 -2.11%
- **실패 원인 3가지**:
  1. 봉 마감 vs 실시간 괴리 — 백테스트는 4시간봉 마감 기준, 실전은 틱 즉시 체결 → 위꼬리 가짜 돌파에 반복 진입
  2. 기간 편향 — 2.2년 평균 승률 38%이지만 최근 6개월 하락장은 26%
  3. 안전장치 지연 — 5연패 자동중단이 4시간 주기 체크라 6연패 후에야 발동

**현재 적용**: **2026-03-29 즉시 중단**. Composite DC20으로 복귀.
- 교훈: `docs/lessons/20260329_1_tick_vs_bar_entry.md`

### 10. VB — 변동성 돌파 (DRY-RUN 검증 중)

**무엇인가?**
래리 윌리엄스의 변동성 돌파를 변형한 일일 단타. 전날의 가격 변동폭을 기준으로, 오늘 가격이 일정 비율 이상 오르면 매수. 하루 만에 청산.

**어떻게 하는가?**
- 매수 레벨 = 오늘 시가 + (어제 고가 - 어제 저가) x K
  - K값은 시장 상태에 따라 조절:
    - 상승장(SMA50 위): K=0.4 (공격적, 작은 움직임에도 진입)
    - 중립: K=0.5
    - 하락장(SMA50 아래): K=0.7 (보수적, 큰 움직임만 진입)
- 매도: 다음날 시가에 자동 청산 / -1.5% 손절
- 최대 3종목 동시 보유

**결과**
- 백테스트(하락장 6개월): 324거래, Sharpe 0.479, MDD -13.8%, 승률 49%, 건당 +0.25%
- DRY-RUN 실전(3/31~4/3, 6건): 승률 33%, 평균 +1.04%, 누적 +6.23%
  - BLUR +12.2% 대박 1건이 전체를 견인
  - 승률 40% 기준 미달 → **7일 연장 (04-07까지)**

**현재 적용**: DRY-RUN (가상매매). VB_ENABLED=True, VB_DRY_RUN=True.
- 04-07에 15건 이상이면 재판정. 승률 40%+ 또는 건당 +1.0%+ 시 실전 전환 검토.

### 11. Vol Reversal — 거래량 반전 (DRY-RUN)

**무엇인가?**
급락 후 거래량이 폭증하면서 반전하는 순간을 잡는 4시간봉 전략. "폭락 + 거래량 폭발 + 양봉 전환"이 동시에 나타나면 바닥 반등으로 판단.

**어떻게 하는가?**
- 매수 조건 (모두 충족):
  1. 거래량 > 20일 평균 x 2.0 (거래량 급증)
  2. RSI(14) < 35 (과매도)
  3. 현재봉 양봉 + 이전봉 음봉 (반전 확인)
- 매도: +3% 익절 / 1.5% 트레일링 / -2% 손절 / 32시간 시간제한

**결과**
- 하락장 6개월 백테스트: 91거래, 승률 56%, 건당 +0.82%, MDD -16.7%
- 월 15회 거래 — Composite(월 1~2회)의 빈도 부족을 보완하는 역할

**현재 적용**: DRY-RUN. `scripts/dryrun_vol_reversal.py` 매 4시간 실행.

### 12. Alt Pump Surf — 알트코인 급등 서핑 (보류)

**무엇인가?**
알트코인이 거래량 4~6배 폭증과 함께 +2% 이상 급등하면 올라타는 전략.

**어떻게 하는가?**
- 매수: 거래량 > SMA(20) x 4.0(상승장)/6.0(하락장) + 가격 +2% + 양봉
- 매도: +3% 익절 / 1.5% 트레일링 / -1% 손절 / 32시간

**결과**
- OOS: Sharpe 0.075, 건당 +0.43~0.74%
- 하락장에서 신호 극소 (6개월 11건) — 쓸 수 없는 수준

**현재 적용**: 보류. 상승장 전환 시 재검토.

---

### 13. RSI Divergence (폐기)

**무엇인가?**
가격은 신저가를 갱신하는데 RSI는 더 높은 저점을 만드는 "다이버전스"를 감지해 매수하는 역추세 전략.

**결과**
- 전 구간, 전 파라미터에서 손실 (Sharpe -0.2 ~ -0.3)
- 업비트 현물(롱온리)에서 역추세 다이버전스는 구조적으로 불리

**현재 적용**: 완전 폐기.

| 거래 횟수 | >= 5건 | >= 3건 |
