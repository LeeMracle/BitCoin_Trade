# ADR: BATA v1.5 — 전략 다각화 + 데이터 확장

> 작성: 2026-04-05 | 상태: **완료** | 04-05 하루 Sprint A~D 15건 전체 완료

## 배경

현재 BATA는 **Composite DC20(추세추종)** 단일 전략으로 실전 운용 중이다.
극공포장(F&G≈10)에서 9건 중 1승 8패, **-16.9% 손실** 발생.

근본 원인 3가지:
1. **데이터 부족** — 백테스트 7.3년(2019~), 2017 광풍/폭락 미포함
2. **전략 편중** — 추세추종만 운용, 횡보/하락장에서 구조적 약점
3. **방어 장치 미흡** — 계좌 레벨 손실 한도(서킷브레이커) 없음

## 결정

**BATA를 v1.5(전략 다각화)로 업그레이드한다.**

- 9년 데이터로 기존 전략 재검증
- 횡보/하락장 보완 전략 1~2종 추가
- 서킷브레이커(계좌 -20% 자동 중단) 설치

## 버전 로드맵

```
v1.0 (과거)   Composite DC20 단일 전략
    ↓
v1.5 (이번)   9년 데이터 재검증 + 신규 전략 분산 + 서킷브레이커
    ↓
v2.0 (향후)   Claude AI 판단 엔진 통합 (KATA v2 참고)
```

## 업그레이드 전/후

```
현재 (v1.0)                            목표 (v1.5)
───────────                            ────────────
데이터: 2019~2026 (7.3년)     →     2017~2026 (9년) + F&G 매크로
전략:   Composite 1종          →     Composite + 신규 1~2종 (분산)
방어:   없음                   →     서킷브레이커 (-20% 자동 중단)
검증:   OOS 2.2년만            →     9년 레짐별 분리 검증
```

## 통과 기준

| 지표 | 엄격 | 완화 (채택 최소) |
|------|------|------------------|
| OOS Sharpe | ≥ 1.0 | ≥ 0.8 |
| OOS MDD | ≥ -20% | ≥ -25% |
| 하락장 구간 Sharpe | ≥ 0.3 | ≥ 0.0 (손실 제한) |
| 거래 횟수 (OOS) | ≥ 10건 | ≥ 5건 |
| 기존 전략과 상관계수 | ≤ 0.5 | ≤ 0.7 |
