## 테스트한 전략 3종

### 전략 1: 변동성 돌파 (Volatility Breakout)

```
원리: Larry Williams 변동성 돌파의 변형
타임프레임: 일봉

작동 방식:
  1. 전일의 고가-저가 = "변동폭" 계산
  2. 당일 시가 + 변동폭 × K = "돌파 기준선"
  3. 종가가 기준선을 넘으면 매수
  4. 다음 날 무조건 매도 (1일 보유)

레짐 적응:
  상승장 (SMA50 위): K=0.4 → 작은 돌파도 진입 (공격적)
  하락장 (SMA50 아래): K=0.7 → 큰 돌파만 진입 (보수적)

왜 작동하는가:
  - 큰 변동이 있는 날, 방향이 정해지면 그 방향으로 더 간다
  - 하락장에서도 반등일에는 변동폭 돌파가 발생
  - 1일만 보유하므로 장기 하락에 노출되지 않음
```

### 전략 2: 알트 펌프 서핑 (Alt Pump Surf)

```
원리: 업비트 알트코인의 "펌프 초기" 감지
타임프레임: 4시간봉

작동 방식:
  1. 거래량이 평소의 4~6배 급증
  2. 가격이 +2% 이상 상승 (양봉)
  3. → "펌프 시작"으로 판단, 매수
  4. +3% 익절 / 1.5% 트레일링 / -1% 손절 / 최대 32시간 보유

왜 가능성이 있는가:
  - 업비트는 개인투자자 중심 → 펌프 패턴이 뚜렷
  - 현물만 있어서 숏 매도 압력이 없음 → 펌프 지속시간이 김
```

### 전략 3: RSI 다이버전스 (RSI Divergence)

```
원리: 가격과 RSI의 괴리가 반전 신호
타임프레임: 4시간봉

작동 방식:
  하락장: 가격은 더 떨어지는데 RSI는 올라감 → "하락 약화" → 매수
  상승장: 가격이 눌리는데 RSI가 더 떨어짐 → "조정 끝" → 매수
```

## 해석 (쉽게)

### 변동성 돌파 — "반만 성공"

**좋은 점:**
- 하락장에서도 수익 (+5.4%, Sharpe 0.479)
- 승률 49% — 거의 반반이지만 이긴 날 수익이 진 날 손실보다 큼
- MDD -13.8% — 기존 composite(-26.4%)보다 방어력 우수
- 거래 빈도 높음 (월 ~54회) — 일봉 전략 대비 50배

**부족한 점:**
- 건당 평균수익 0.11~0.25% — 수수료(0.1%) 빼면 남는 게 적음
- Sharpe 0.253 — 목표 0.6에 미달 (수익/위험 비율 부족)
- 파라미터 최적화로 개선 여지는 있음

**비유하면:** 매일 동전 던지기를 하는데, 이기면 200원, 지면 150원 잃는 게임.
장기적으로 돈은 벌지만, 한 판의 수익이 작아서 수수료 부담이 큼.

### 알트 펌프 서핑 — "신호가 너무 적음"

**좋은 점:**
- 건당 수익 0.43~0.74% — 수수료 후에도 충분
- MDD가 매우 작음 (-3~5%)

**부족한 점:**
- 하락장에서 거래 11건 / 6개월 → 통계적 의미 없음
- 하락장 승률 11~38% → 작동하지 않음

**비유하면:** 아주 좋은 기회만 잡는 전략인데, 하락장에서는 그런 기회가 거의 없음.

### RSI 다이버전스 — "폐기"

- 모든 구간, 모든 파라미터에서 손실
- 업비트 현물(롱온리)에서는 구조적으로 부적합
- **더 이상 시간 투자 가치 없음**
