### 운용 환경

- 4시간마다 레벨 갱신 (UTC 0/4/8/12/16/20시)
- 최대 동시 5포지션, 균등 배분
- 최소 주문금액: 5,000원 (업비트 최소)

## 3. 검증 플랜

### 목적
소액(26만원)으로 전략의 실전 승률을 검증한다.
수익금보다 매매 횟수가 중요 — 수수료 대비 수익은 무시한다.

### 판단 기준

| 시점 | 최소 거래 | 조치 |
|------|:--------:|------|
| 3일 (3/31) | 5건+ | 전패(0승)면 즉시 중단 |
| 5일 (4/2) | 10건+ | 승률 30% 미만이면 전략 수정 |
| 7일 (4/4) | 15건+ | 승률 35%+면 증액 검토, 미달이면 전략 변경 |

### 조기 탈출
- 5건 연속 손실 → 즉시 중단, 원인 분석
- 10건 시점 승률 20% 미만 → 전략 수정

### 성공 기준 (백테스트 대비)
- 승률: 38% (백테스트) → 실전 35% 이상이면 합격
- 평균수익: +0.91% (백테스트) → 실전 +0.5% 이상이면 합격

1. **전략 전환은 최소 7일 단위** — 하루 만에 바꾸지 않는다
2. 현재 계좌(26만원)는 "수업료" — 검증 완료 후 별도 자금 투입
3. 전략 변경은 주말에 검토, 평일에는 데이터 수집에 집중
4. 매매 발생 시 즉시 텔레그램 알림 + 4시간 정기 보고에 누적 성적 포함
