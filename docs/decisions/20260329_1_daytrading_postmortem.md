# 20260329_1: daytrading 전략 사후 분석 및 중단

## 결정
daytrading(DC15+Vol2.5x+Trail2%) 즉시 중단, composite DC20으로 복귀

## 실전 성적
- 8건 거래: 1승 7패 (12.5%)
- 평균: -2.11%, 6연패
- KNC -10.24% (전략전환 버그)

## 실패 원인
1. **실시간 틱 진입 vs 봉 마감 진입 괴리**: 백테스트는 봉 마감 기준, 실전은 틱 즉시 체결 → 가짜 돌파에 진입
2. **하락장 구조적 한계**: 최근 6개월 백테스트 26% 승률 (2.2년 38%는 상승장 포함)
3. **5연패 자동 중단 미작동**: 4시간 보고 주기에만 체크, 사이에 연패 발생

## 교훈
- 4시간봉 전략은 반드시 4시간봉 마감 후 진입 (실시간 틱 금지)
- 백테스트 기간에 상승장 비중이 높으면 하락장 성과 과대평가
- 전략 전환 시 QA 검증 필수
