### P5-07 BTC 필터 + F&G 레짐 스위치 리서치 (04-04 ~ 04-05)

현재 문제: 극공포장(F&G 11)에서도 봇이 돌파 신호를 추종 → 연패.

**리서치 항목 6가지:**

| ID | 아이디어 | 가설 | 백테스트 방법 |
|----|---------|------|-------------|
| A | BTC 필터 | 알트 매수 전 BTC > SMA(20) 확인 | composite에 조건 추가 후 OOS 비교 |
| B | 시간대 필터 | 09:00~15:00 KST 돌파만 유효 | timestamp 기반 필터 추가 |
| C | 연패 쿨다운 | 3연패 시 3일 매수 금지 | bar-by-bar 루프에 상태 추가 |
| D | ATR 포지션 사이징 | 변동성 높으면 소액, 낮으면 대량 | 신호 0/1 → 연속값으로 확장 |
| E | 코인 섹터 분산 | 같은 섹터 2종목 이상 진입 차단 | multi_trader 레벨 필터 |
| F | F&G 레짐 스위치 | F&G < 20이면 매매 중단 | fetch_fear_greed 데이터 연동 |

**우선순위**: F(F&G 스위치) → A(BTC 필터) → C(연패 쿨다운) → 나머지

**성공 기준**: 기존 composite OOS 대비
- MDD 개선 (현 -26.4% → -20% 이하)
- Sharpe 유지 또는 개선 (현 1.28)
- 하락장 구간 손실 감소
