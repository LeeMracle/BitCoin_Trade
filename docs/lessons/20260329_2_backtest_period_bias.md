# 백테스트 기간에 따른 성과 과대평가

- **발생일**: 2026-03-29
- **심각도**: HIGH
- **카테고리**: 전략

## 증상

daytrading 전략이 2.2년 백테스트에서 38% 승률, +0.91% 평균 수익이었으나,
최근 6개월(하락장) 백테스트에서는 26% 승률, 마이너스 수익.
실전에서도 12.5% 승률로 참패.

## 원인

- 2.2년 백테스트 기간에 2024년 상승장(BTC +300%)이 포함
- 상승장 비중이 높아 전체 성과가 과대평가됨
- 하락장(BTC ATH 대비 -41%, F&G 10)에서 롱온리 단타는 구조적 불리

## 수정

`docs/decisions/20260329_daytrading_stop_composite_switch.md` — daytrading 즉시 중단, composite DC20 전환.

향후 전략 평가 시:
1. 전체 기간 OOS 성과 + **최근 6개월 하락장 구간별 성과** 분리 확인
2. 하락장 구간에서 양수 수익 또는 MDD -10% 이내가 아니면 실전 투입 금지

## 검증규칙

전략 신규 투입 전 체크리스트:
- [ ] 전체 OOS Sharpe >= 0.8
- [ ] 최근 6개월 하락장 구간 별도 검증
- [ ] 하락장 MDD >= -15%
- [ ] 승률 >= 35% (하락장 구간)

## 교훈

백테스트 기간이 상승장을 많이 포함하면 하락장 성과를 과대평가한다.
전략 투입 전 반드시 최근 하락장 구간을 분리하여 별도 검증할 것.
"전체 기간 OK"는 "지금 OK"를 의미하지 않는다.
