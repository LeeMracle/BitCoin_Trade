# 레짐 필터 비교 백테스트 보고서

생성일: 2026-04-05

## 개요

Composite DC20 전략에 3종 레짐 필터를 적용하여 성과를 비교한다.
CRISIS 레짐(F&G 극공포, 하락장)에서 손실을 줄이는 것이 목적.

**기간 정의**

| 구간 | 시작 | 종료 |
|------|------|------|
| 워밍업 | 2017-10-01 | 2018-05-31 |
| IS (인샘플) | 2018-06-01 | 2023-12-31 |
| OOS (아웃오브샘플) | 2024-01-01 | 2026-04-04 |

**필터 정의**

| 필터 | 설명 |
|------|------|
| Baseline | 필터 없음 (DC20 원본) |
| Filter A | F&G < 20이면 신규 진입 금지 (기존 포지션 유지) |
| Filter B | BTC 종가 < 200EMA이면 신규 진입 금지 |
| Filter C | F&G < 20 → 진입 0%, F&G < 30 → 진입 50% (사이즈 축소) |

## 필터링 효과 분석 (OOS 구간)

필터에 의해 '차단된 거래'가 실제로 나쁜 거래였는지 확인한다. 필터가 효과적이라면 차단된 거래의 승률이 낮고 평균 수익이 음수여야 한다.

기준선 OOS 총 거래수: **11건**

| 필터 | 차단된 거래수 | 차단 거래 승률 | 차단 거래 평균수익 | 필터 유효성 |
|------|--------------|----------------|-------------------|-------------|
| Filter A | 1건 | 0.0% | -4.44% | 유효 (나쁜 거래 차단) |
| Filter B | 2건 | 0.0% | -3.87% | 유효 (나쁜 거래 차단) |
| Filter C | 1건 | 0.0% | -4.44% | 유효 (나쁜 거래 차단) |

### 필터 A 상세 (F&G < 20 차단)

- OOS 구간 차단일: 82일 (전체 OOS 중)
- OOS F&G 분포: < 20 = 82일, 20~30 = 124일, >= 30 = 618일

### 필터 B 상세 (BTC < 200EMA 차단)

- OOS 구간 차단일: 236일 (전체 OOS 중)
- OOS BTC < 200EMA 일수: 236일

### 필터 C 상세 (F&G 기반 사이즈 축소)

- OOS 사이즈=0% (F&G<20): 82일
- OOS 사이즈=50% (20<=F&G<30): 124일
- OOS 사이즈=100% (F&G>=30): 619일
- 참고: BacktestEngine은 사이즈 축소를 직접 지원하지 않으므로 사이즈 50% 구간의 거래는 기준선과 동일하게 처리되고, 사이즈 0% 구간(F&G<20)만 진입이 차단된다.
