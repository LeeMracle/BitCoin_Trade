## 구조적 진단

### F&G 역추세 전략의 근본 한계

```
문제 1: 강세장에서 대부분 현금
  - F&G ≤20 → 극도의 공포 시에만 진입
  - 2020~2021 강세장 대부분을 놓침 (BnH +1446% vs 전략 -40%)

문제 2: 역추세 ≠ 추세추종
  - BTC는 장기 추세가 강한 자산 (모멘텀 프리미엄)
  - 역추세는 조정 국면의 반등만 포착 → 대세 상승의 일부만 획득

문제 3: 거래 빈도 부족
  - F&G ≤20 이벤트: 5년간 8회 (평균 7.5개월 간격)
  - 통계적 유의성 확보 불가능한 표본 크기

문제 4: 비대칭 리스크
  - 강세장 조정: 빠른 반등 → 수익 (OK)
  - 약세장 진입: 추가 하락 → 손절 연쇄 (FAIL)
  - 실시간으로 두 국면 구별 불가
```
